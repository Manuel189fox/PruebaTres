Prueba Hip√≥tesis

Se indica la prueba hip√≥tesis para la variable ‚Äúmath score‚Äù:

\[ \begin{matrix} H_{0}:\mu =60\\ H_a:\mu \neq 60 \end{matrix} \]

#alternative:"greater", "less" or "two.sided" 
#seg√∫n corresponda la prueba alterna de la hipotesis Ha.
test1=z.test(x=df.m$score.marh,
              sigma.x = sd(df.m$score.marh),
              mu=60,
              alternative="two.sided",
              conf.level = 0.95)

Prueba Hip√≥tesis: Conclusi√≥n

Podemos concluir que existe evidencia suficiente para Rechaza \(H_{0}\), la media de la variable ‚Äúmath score‚Äù no es igual a 65.

El intervalo de confianza nos indica que existe diferencia entre las medias, pues presentan el mismo signo.

Prueba Hip√≥tesis para dos muestras

A continuaci√≥n se presenta la prueba hip√≥tesis para la direfencias de medias de: \(\mu_{score.marh}\) y \(\mu_{score.reading}\)

\[ \begin{matrix} H_{0}:\mu_{1}-\mu_{2} =0\\ H_{0}:\mu_{1}-\mu_{2} \neq 0 \end{matrix} \]

test2=z.test(x=df.m$score.marh,
             y=df.r$score.reading,
             sigma.x = sd(df.m$score.marh),
             sigma.y = sd(df.r$score.reading),
             mu=0,
             alternative="two.sided",
                conf.level = 0.95)

Para un nivel de significancia del 5%, podermos afirmar que el valor del Z estad√≠stico es del: -2.2037362, adem√°s el p-valor es de: 0.0275429.
